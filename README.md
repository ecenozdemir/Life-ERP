# 🚀 Life-ERP: Kişisel Gelişim ve Analiz Sistemi

Bu proje, bir yazılımcı ve veri analisti adayının kendi hayatını bir **ERP sistemi** gibi yönetme, veriye dökme ve analiz etme yolculuğudur.

## 📊 Projenin Amacı
Kendi biyolojik döngülerim (PMS/Ovülasyon), dışsal faktörler (Dolar kuru, hava durumu) ve üretkenliğim (kod satırı, kitap sayfası, sosyal temas) arasındaki gizli bağları keşfetmek.

## 🛠️ Kullanılan Teknolojiler
* **Dil:** Python 3.x
* **Veritabanı:** SQLite3
* **Versiyon Kontrol:** Git & GitHub
* **Otomasyon:** `requests` kütüphanesi ile anlık veri çekme

## 📈 Otomatik Takip Edilen Veriler (Live APIs)
* ✅ **Dolar Kuru:** Her veri girişinde anlık olarak güncellenir.
* ✅ **Hava Durumu:** Konum bazlı (İstanbul) derece ve durum bilgisi çekilir.

## ✍️ Günlük Takip Edilen Manuel Veriler
* **Mood ve Enerji Puanı (1-10)**
* **Sosyal Temas / İletişim (1-10)**
* **Yazılan Kod Satırı**
* **Okunan Kitap Sayfası**
