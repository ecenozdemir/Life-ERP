# 🚀 Life-ERP: Kişisel Gelişim ve Analiz Sistemi

Bu proje, bir yazılımcı ve veri analisti adayının kendi hayatını bir **ERP sistemi** gibi yönetme, veriye dökme ve analiz etme yolculuğudur.

## 📊 Projenin Amacı
Kendi biyolojik döngülerim (PMS/Ovülasyon), dışsal faktörler (Dolar kuru, hava durumu, ev kalabalığı) ve üretkenliğim (kod satırı, kitap sayfası) arasındaki gizli bağları keşfetmek.

## 🛠️ Kullanılan Teknolojiler
* **Dil:** Python 3.x
* **Veritabanı:** SQLite3
* **Versiyon Kontrol:** Git & GitHub

## 📝 Günlük Takip Edilen Veriler
* Mood ve Enerji Puanı (1-10)
* Yazılan Kod Satırı
* Okunan Kitap Sayfası
* Uyku Süresi
* *Yakında:* Otomatik Dolar Kuru ve Hava Durumu Entegrasyonu!
